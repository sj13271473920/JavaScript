### 控件的滚动，遍历控件并操作  数据类型
```
Listview	//列表容器
widget.RecyclerView	//列表容器
ScrollView	//滚动容器
scrollForward()	//对控件执行向前滑动的操作,并返回是否操作成功。
scrollBackward()	//对控件执行向后滑动的操作,并返回是否操作成功。
findOne	//找到一个	find	//找出所有的
length	//返回数组的数目
bounds	//返回控件的坐标
centX()	//返回对象的x坐标	centY()	//返回对象的y坐标



```
#### 案例
```
className("ListView").scrollForward()
```
